# 🖱️ Virtual Hand Mouse

Control your computer mouse using only your hand!

This project uses Computer Vision and Hand Tracking to move the mouse cursor with your index finger.

## ✨ Features

- 👆 Move the mouse with your index finger
- 🎯 Smooth mouse movement
- 📷 Real-time webcam tracking
- ⚡ FPS counter
- 🖐️ Hand tracking using MediaPipe

## 🛠️ Technologies

- Python
- OpenCV
- cvzone
- MediaPipe
- PyAutoGUI

## 📦 Installation

```bash
pip install opencv-python
pip install cvzone
pip install mediapipe
pip install pyautogui
```

## 🚀 Run

```bash
python magic_mouse.py
```
